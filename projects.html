Research

Toxicity Prediction Using Graph Neural Networks

Aug 2024 - Sep 2025

Problem

The goal was to improve small-molecule toxicity prediction so that molecular screening could become both more accurate and more interpretable.

Data

Worked on public benchmark datasets containing molecular structures and toxicity labels. The modeling pipeline also incorporated atom-level features, functional group information, and fingerprint-style descriptors.

Approach

Built and evaluated multiple graph neural network architectures, then enhanced the models by integrating atom, functional group, and fingerprint features with functional group importance analysis.

Outcome / Impact

The project improved both model performance and interpretability across benchmark experiments, and supported one conference paper plus a follow-up journal manuscript under revision.

Your Contribution

I designed and ran the model experiments, organized the feature pipeline, analyzed functional-group importance, and supported the writing and interpretation of the resulting research outputs.

Research

Microscopic Image Classification of Bacteria

Dec 2020 - Feb 2021

Problem

The project aimed to automate classification of microscopic bacterial images in order to reduce manual workload and improve classification efficiency.

Data

Used a labeled microscopic image dataset covering five bacteria categories. The main input was image-based visual data prepared for supervised CNN classification.

Approach

Designed and trained deep learning models in Python, wrote the model architectures, and tuned parameters and network settings to compare candidate CNN approaches.

Outcome / Impact

The final SE-ResNet-based model achieved over 98% accuracy across the five bacterial categories, showing strong potential for reliable automated image classification.

Your Contribution

I was responsible for model design, training, parameter tuning, architecture comparison, and summarizing the final result that justified choosing SE-ResNet.

Development

Hemodialysis Health Education App

Dec 2019 - Dec 2020

Problem

The app was built to reduce the health education burden on clinicians and improve the quality of home-care education for patients and families.

Data

The project involved patient-education content, app-side database records, and clinical workflow requirements that shaped both the data structure and the system flow.

Approach

I participated in system analysis, database design, C# Windows-side development, Android mobile development, and the implementation of data transfer across the system.

Outcome / Impact

The app was launched in Tzu-Chi Hospital, and satisfaction from patients and doctors reached 4.8 / 5.0, showing practical value in a real care setting.

Your Contribution

I worked across requirement research, function definition, database and app design, programming, and technical support so that the system could move from concept to hospital use.
